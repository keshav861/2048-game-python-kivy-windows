2048 Game
---------
Version: 1.0.0

Installation:
1. Extract all files to any folder
2. Double-click 2048.exe to run
3. No installation needed

Requirements:
- Windows 7/8/10/11
- 64-bit system

Controls:
- Arrow keys or WASD to move tiles
- ESC to exit
- R to restart

Note: Keep all files in the same folder structure as provided.

Enjoy playing!
